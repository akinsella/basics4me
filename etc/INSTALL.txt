===========
Using Basics4ME
===========
 
Please, refer to following wiki related page: http://wiki.helyx.org/basics4me
 
=========
JAR files
=========

The basics4me distribution comes with one jar file: basics4me-VERSION.jar
under the BASICS4ME_HOME directory.

This jar file contains all the class files of the basics4me project,
except test cases packages.

==============
Building basics4me
==============

basics4me is built with Maven 2. To rebuild basics4me-VERSION.jar,
place Maven 2 on the PATH and execute "mvn package".    


In case of problems send an e-mail note to: dev@helyx.org.
